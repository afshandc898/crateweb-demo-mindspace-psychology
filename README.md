# MindSpace Psychology — Demo Website

A CrateWeb sales demo for allied health prospects.

## Tech Stack
- React + Vite
- Tailwind CSS v4

## Development
```bash
npm install
npm run dev
```

## Build & Deploy
```bash
npm run build
```

### Cloudflare Pages
1. Connect this repo to Cloudflare Pages (or use `wrangler pages deploy dist/`)
2. Build command: `npm run build`
3. Build output directory: `dist`
4. Node version: 18+
